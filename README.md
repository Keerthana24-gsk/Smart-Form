# Smart Form – Voice Interactive Form Filling Assistant

## 📌 Project Description
Smart Form is a voice-based interactive system that automates online form filling using speech recognition, Natural Language Processing (NLP), and browser automation. The system allows users to complete web forms through voice interaction without relying on a keyboard or mouse, improving accessibility and user convenience.

The project is designed to assist visually impaired users, elderly users, and individuals who face difficulty with conventional form filling methods.

---

## ✨ Features
- Voice-based form filling
- Speech-to-text conversion
- NLP-based information extraction
- Automatic form field mapping
- Browser automation using Selenium
- Voice confirmation before submission
- Hands-free user interaction

---

## 🛠 Technologies Used
- Python
- Flask
- Selenium
- spaCy (NLP)
- Google Speech Recognition
- pyttsx3
- HTML
- CSS
- JavaScript

---

## ⚙️ System Workflow
1. User provides voice input
2. Speech is converted into text
3. NLP extracts meaningful information
4. System identifies corresponding form fields
5. Selenium automatically fills the form
6. Voice confirmation is provided
7. Form is submitted after user confirmation
